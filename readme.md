# Estudo de desenvolvimento em Python

Este projeto contém exercícios de desenvolvimento propostos pela faculdade de [data science Uninter.](https://www.uninter.com/)

Os exercícios com as aplicações foram separados e nomeados nos arquivos, porém há arquivos onde todos estão misturados pois foram
aplicados durante a aula. Então fica mais fácil ver a aplicação separada no seu arquivo específico mas, é possível vê-los nos arquivos
maiores para checar a linha de racícionio.

Os arquivos maiores estão nomeados com "teste_(nome da função/comando)", já os arquivos específicos estão com o nome da sua
aplicação.

Alguns conteúdos abordados:

* Hello world!;
* Função de saída (print);
* Operadores e operações matemáticas;
* Variáveis, dados, seus tipos e composição (numérica, lógica...);
* Strings (concatenação, fatiamento, função Len);
* Função de entrada (input);
* Função condicional simples e composta;
